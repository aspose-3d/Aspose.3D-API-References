## Discreet3dsSaveOptions.HighPreciseColor property

これが true の場合、生成された 3ds ファイルは高精度の色を使用します。これは、赤/緑/青の各チャネルが 32 ビット float であることを意味します。 そうでない場合、生成されたファイルは 24 ビット色を使用し、各チャネルは 8 ビット バイトを使用します。すべてのアプリケーションが高精度の色をサポートしているわけではないため、false です。

public bool HighPreciseColor { get; set; }
