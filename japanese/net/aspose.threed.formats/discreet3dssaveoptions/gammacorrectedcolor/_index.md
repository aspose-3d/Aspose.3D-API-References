## Discreet3dsSaveOptions.GammaCorrectedColor property

3ds ファイルには、同じ属性の元の色とガンマ補正された色が含まれる場合があります。 これを true に設定すると、可能であればガンマ補正された色が使用されます。 そうでない場合、Aspose.3D は元の色を使用しようとします。

public bool GammaCorrectedColor { get; set; }
