## Discreet3dsSaveOptions.DuplicatedNameSeparator property

オブジェクトの名前と複製されたカウンターの間のセパレーター。デフォルト値は「_」です。 シーンに同じ名前を使用するオブジェクトが含まれている場合、Aspose.3D 3DS エクスポーターはオブジェクトに別の名前を生成します。 たとえば、2 つのノードがあります。最初のノードは "Box", という名前になり、2 番目のノードはデフォルト構成を使用して新しい名前 "Box_2" を取得します.

```csharp
public string DuplicatedNameSeparator { get; set; }
```
