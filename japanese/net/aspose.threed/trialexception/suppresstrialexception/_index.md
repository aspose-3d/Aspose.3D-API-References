## TrialException.SuppressTrialException property

これを true に設定して、ライセンスのない使用に対する試用例外を抑制しますが、制限は解除されません. 制限を解除するには、適切なライセンスを使用してください. また、これを true に設定すると、ライセンスのない使用を認識していることを意味します制限.

public static bool SuppressTrialException { get; set; }
