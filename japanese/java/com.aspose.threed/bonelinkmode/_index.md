## フィールド

| フィールド | 説明 |
| --- | --- |
| [ADDITIVE](#ADDITIVE) | 加算モードは、子ボーンのローカル変換を親ボーンの変換に加えることで、子ボーンの変換を計算します。 |
| [NORMALIZE](#NORMALIZE) | このモードでは、子ボーンの変換は親ボーンの変換に対して正規化されます。 |
| [TOTAL_ONE](#TOTAL-ONE) | トータルワンは、親ボーンと子ボーンの合成変換が全体で1単位の長さにスケールするようにします。 |
