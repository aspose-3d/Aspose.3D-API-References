## Discreet3dsSaveOptions.HighPreciseColor property

Jika ini benar, file 3ds yang dihasilkan akan menggunakan warna presisi tinggi, artinya setiap saluran merah/hijau/biru berada dalam float 32bit. Jika tidak, file yang dihasilkan akan menggunakan warna 24bit, setiap saluran menggunakan byte 8bit. Nilai default salah, karena tidak semua aplikasi mendukung warna presisi tinggi.
