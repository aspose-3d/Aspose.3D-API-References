Ruang nama animasi dari Aspose.3D, semua kelas terkait animasi ditentukan dalam ruang nama ini

## Kelas

| Kelas | Keterangan |
| --- | --- |
| [AnimationChannel](./animationchannel/) | Saluran memetakan bidang komponen properti ke serangkaian rangkaian keyframe |
| [AnimationClip](./animationclip/) | Klip animasi adalah kumpulan animasi. Adegan dapat memiliki satu atau beberapa klip animasi. |
| [AnimationNode](./animationnode/) | Aspose.3D mendukung hierarki animasi, setiap animasi dapat disusun oleh beberapa animasi dan definisi key-frame animasi. [`AnimationNode`](../aspose.threed.animation/animationnode/) mendefinisikan transformasi nilai properti dari waktu ke waktu, misalnya node animasi dapat digunakan untuk mengontrol transformasi node atau lainnya[`A3DObject`](../aspose.threed/a3dobject/) properti numerik objek. |
| [BindPoint](./bindpoint/) | A[`BindPoint`](../aspose.threed.animation/bindpoint/)biasanya dibuat pada properti objek, beberapa tipe properti berisi beberapa bidang komponen (seperti bidang Vector3), [`BindPoint`](../aspose.threed.animation/bindpoint/) akan menghasilkan saluran untuk setiap bidang komponen dan menghubungkan bidang ke satu atau beberapa instance urutan bingkai utama melalui saluran. |
| [Extrapolation](./extrapolation/) | Ekstrapolasi menentukan bagaimana melakukan ketika nilai sampel berada di luar rentang yang ditentukan oleh bingkai kunci pertama dan terakhir. |
| [KeyFrame](./keyframe/) | Bingkai kunci terutama ditentukan oleh waktu dan nilai, untuk beberapa jenis interpolasi, garis singgung/ketegangan/bias/kontinuitas juga digunakan dengan menghitung nilai sampel akhir. Nilai sampel dalam posisi waktu non-kerangka kunci diinterpolasi dengan key-frame antara key-frame sebelumnya dan berikutnya Nilai sebelum/setelah key-frame pertama/terakhir dihitung dengan[`Extrapolation`](../aspose.threed.animation/extrapolation/) kelas. |
| [KeyframeSequence](./keyframesequence/) | Urutan bingkai kunci, yang menjelaskan transformasi nilai sampel dari waktu ke waktu. |
