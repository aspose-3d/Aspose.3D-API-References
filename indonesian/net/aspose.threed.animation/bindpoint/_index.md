## BindPoint class

A`BindPoint`biasanya dibuat pada properti objek, beberapa tipe properti berisi beberapa bidang komponen (seperti bidang Vector3), `BindPoint` akan menghasilkan saluran untuk setiap bidang komponen dan menghubungkan bidang ke satu atau beberapa instance urutan bingkai utama melalui saluran.
