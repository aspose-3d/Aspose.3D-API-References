## RelativeRectangle class

Persegi panjang relatif  Rumus antara komponen relatif ke nilai absolut adalah:  Skala  (Lebar Referensi) + offset  Jadi jika kita ingin itu merepresentasikan nilai absolut, biarkan semua bidang skala nol, dan gunakan bidang offset sebagai gantinya.
