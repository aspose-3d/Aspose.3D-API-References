## VertexElementMaterial class

Mendefinisikan indeks material untuk komponen yang ditentukan.  Sebuah node dapat memiliki beberapa material, VertexElementMaterial digunakan untuk merender bagian berbeda dari geometri dengan material yang berbeda.
