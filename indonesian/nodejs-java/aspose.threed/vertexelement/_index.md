## VertexElement class

Kelas dasar elemen vertex.  Tipe elemen vertex diidentifikasi oleh VertexElementType.  Sebuah VertexElement menjelaskan bagaimana elemen vertex dipetakan ke permukaan geometri dan bagaimana informasi pemetaan diatur dalam memori.  Sebuah VertexElement berisi Normals, UVs, atau jenis informasi lainnya.  @hideconstructor
