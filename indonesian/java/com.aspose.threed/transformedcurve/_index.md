Sebuah [TransformedCurve](../../com.aspose.threed/transformedcurve) memberikan penempatan pada kurva dengan menggunakan matriks transformasi. Ini memungkinkan melakukan transformasi di dalam sebuah [TrimmedCurve](../../com.aspose.threed/trimmedcurve) atau [CompositeCurve](../../com.aspose.threed/compositecurve).
