```
public class Text extends Profile
```

Textprofil, den här profilen beskriver konturer med hjälp av teckensnitt och text. **Exempel:** Följande kod visar hur man skapar ett 3D-nät från text med en angiven teckensnittsfil.

```
var font = FontFile.fromFile("CascadiaCode-Regular.otf");
             var text = new Text();
             text.setFont(font);
             text.setContent("ABC");
             text.setFontSize(10);
             var linear = new LinearExtrusion(text, 10).toMesh();
             var scene = new Scene(linear);
             scene.save("test.stl");
```
