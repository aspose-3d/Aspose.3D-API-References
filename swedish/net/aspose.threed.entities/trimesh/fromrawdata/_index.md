## TriMesh.FromRawData method

Skapa TriMesh från rådata

```csharp
public static TriMesh FromRawData(VertexDeclaration vd, byte[] vertices, int[] indices, 
    bool generateVertexMapping)
```

| Parameter | Typ | Beskrivning |
| --- | --- | --- |
| vd | VertexDeclaration | Vertex-deklaration, måste innehålla minst ett fält. |
| vertices | Byte[] | Inmatade vertexdata, minsta längd på hörnen måste vara större eller lika med vertexdeklarationens storlek |
| indices | Int32[] | Triangelindexen |
| generateVertexMapping | Boolean | Generera[`Vertex`](../../../aspose.threed.utilities/vertex) för varje vertex, vilket inte är nödvändigt för bara serialisering/deserialisering. |

### Returvärde

De[`TriMesh`](../../trimesh) instans som kapslade in bytearrayen.

### Anmärkningar

Den returnerade TriMesh kommer inte att kopiera indatabyte-arrayen för prestanda, externa ändringar på arrayen kommer att återspeglas i den här instansen.
