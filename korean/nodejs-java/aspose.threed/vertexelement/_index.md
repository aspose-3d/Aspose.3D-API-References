## VertexElement class

정점 요소의 기본 클래스.  정점 요소 유형은 VertexElementType으로 식별됩니다.  VertexElement는 정점 요소가 기하학 표면에 어떻게 매핑되는지와 매핑 정보가 메모리에 어떻게 배치되는지를 설명합니다.  VertexElement는 Normals, UVs 또는 기타 종류의 정보를 포함합니다.  @hideconstructor
