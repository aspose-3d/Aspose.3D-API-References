## Discreet3dsSaveOptions.DuplicatedNameSeparator property

개체 이름과 복제된 카운터 사이의 구분 기호, 기본값은 "_"입니다. 장면에 동일한 이름을 사용하는 개체가 포함된 경우 Aspose.3D 3DS 내보내기는 개체에 대해 다른 이름을 생성합니다. 예를 들어 두 개의 노드가 있습니다. 이름이 "Box"인 경우 첫 번째 노드의 이름은 "Box", 이고 두 번째 노드는 기본 구성을 사용하여 "Box_2"라는 새 이름을 갖게 됩니다.

public string DuplicatedNameSeparator { get; set; }
