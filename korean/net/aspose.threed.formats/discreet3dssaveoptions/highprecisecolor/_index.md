## Discreet3dsSaveOptions.HighPreciseColor property

이것이 참이면 생성된 3ds 파일은 고정밀 색상을 사용합니다. 즉, 빨강/녹색/파랑의 각 채널이 32비트 플로트에 있음을 의미합니다. 그렇지 않으면 생성된 파일은 24비트 색상을 사용하고 각 채널은 8비트 바이트를 사용합니다. 기본값 모든 응용 프로그램이 고정밀 색상을 지원하는 것은 아니기 때문에 거짓입니다.

public bool HighPreciseColor { get; set; }
