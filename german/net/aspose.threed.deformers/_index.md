## Klassen

| Klasse | Beschreibung |
| --- | --- |
| [Bone](./bone) | Ein Bone definiert die Teilmenge des Kontrollpunkts der Geometrie und das definierte Mischungsgewicht für jeden Kontrollpunkt. Die[`Bone`](../aspose.threed.deformers/bone) Objekt kann nicht direkt verwendet werden, a[`SkinDeformer`](../aspose.threed.deformers/skindeformer) Instanz wird verwendet, um die Geometrie zu verformen, und[`SkinDeformer`](../aspose.threed.deformers/skindeformer)wird mit einer Reihe von Bones geliefert, wobei jeder Bone mit einem Knoten verknüpft ist. HINWEIS: Ein Kontrollpunkt einer Geometrie kann an mehr als einen Bones gebunden werden. |
| [Deformer](./deformer) | Basisklasse für[`SkinDeformer`](../aspose.threed.deformers/skindeformer) und[`MorphTargetDeformer`](../aspose.threed.deformers/morphtargetdeformer) |
| [MorphTargetChannel](./morphtargetchannel) | Ein MorphTargetChannel wird von verwendet[`MorphTargetDeformer`](../aspose.threed.deformers/morphtargetdeformer) um die Zielgeometrien zu organisieren. Einige Dateiformate wie FBX unterstützen mehrere Kanäle parallel. |
| [MorphTargetDeformer](./morphtargetdeformer) | MorphTargetDeformer bietet Animationen pro Scheitelpunkt. MorphTargetDeformer organisiert alle Ziele über[`MorphTargetChannel`](../aspose.threed.deformers/morphtargetchannel) , kann jeder Kanal mehrere Ziele organisieren. Eine häufige Verwendung des Morph-Zielverformers ist das Anwenden von Gesichtsausdrücken auf eine Figur. Weitere Einzelheiten finden Sie unter https://en.wikipedia.org/wiki/Morph_target_animation |
| [SkinDeformer](./skindeformer) | Ein Skin-Deformer enthält mehrere zu bearbeitende Bones, jeder Bone mischt einen Teil der Geometrie durch die Gewichtung der Kontrollpunkte. |
