## TrialException.SuppressTrialException property

Setzt dies auf true, um die Testausnahme für unlizenzierte Nutzung zu unterdrücken, aber die Beschränkungen werden nicht aufgehoben. Um die Beschränkungen aufzuheben, verwenden Sie bitte eine ordnungsgemäße Lizenz. Und setzt dies auf true, bedeutet auch, dass Sie sich der unlizenzierten Nutzung bewusst sind Einschränkungen.

public static bool SuppressTrialException { get; set; }
