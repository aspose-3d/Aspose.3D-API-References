## ApertureMode enumeration

Kamerablendenmodi. Der Blendenmodus bestimmt, welche Werte die Kamerablende steuern. Wenn der Blendenmodus HorizAndVert, Horizontal oder Vertical ist, wird das Sichtfeld verwendet. Wenn der Blendenmodus FocalLength ist, wird die Brennweite verwendet.

```csharp
public enum ApertureMode
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| HorizAndVert | `0` | Legen Sie die Winkelwerte für die horizontalen und vertikalen Einstellungen fest. |
| Horizontal | `1` | Stellen Sie nur den horizontalen Winkel ein. |
| Vertical | `2` | Stellen Sie nur den vertikalen Winkel ein. |
| FocalLength | `3` | Brennweite direkt verwenden. |
