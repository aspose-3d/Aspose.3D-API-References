## MappingMode enumeration

Legt fest, wie das Element auf eine Fläche abgebildet wird. Die[`MappingMode`](../mappingmode) definiert wie[`VertexElement`](../vertexelement) wird auf die Oberfläche der Geometrie abgebildet.

```csharp
public enum MappingMode
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| ControlPoint | `0` | Alle Daten werden dem Kontrollpunkt der Geometrie zugeordnet. |
| PolygonVertex | `1` | Die Daten werden dem Scheitelpunkt des Polygons zugeordnet Wenn ein Kontrollpunkt von mehreren Polygonen gemeinsam genutzt wird und die Daten zugeordnet werden alsPolygonVertex der Kontrollpunkt als unterschiedlicher Polygonscheitel hat seine eigenen data |
| Polygon | `2` | Die Daten werden dem Polygon zugeordnet. Jeder Polygon-Scheitelpunkt nutzt im Mapping-Modus dieselben DatenPolygon . |

| AllSame | `4` | Eine Datenmenge, die auf die gesamte Oberfläche abgebildet wird. Welche Daten auch immer als Kontrollpunkt/Polygonscheitelpunkt/Kantenendpunkt interpretiert werden, die Daten sind immer die gleichen wie sie definiert wurdenAllSame . |
