## VertexElement class

Basisklasse von Scheitelpunktelementen. Ein Scheitelpunktelementtyp wird durch VertexElementType identifiziert. Ein VertexElement beschreibt, wie das Vertexelement auf eine Geometrieoberfläche abgebildet wird und wie die Abbildungsinformationen im Speicher angeordnet sind.
