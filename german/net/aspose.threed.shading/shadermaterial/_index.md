## ShaderMaterial class

Ein Shader-Material ermöglicht die Beschreibung des Materials durch eine externe Rendering-Engine oder Shader-Sprache. [`ShaderMaterial`](../shadermaterial) Verwendet[`ShaderTechnique`](../shadertechnique)Um die konkreten Rendering-Details zu beschreiben, wird verwendet, und je nach endgültiger Rendering-Plattform wird die am besten geeignete verwendet. Zum Beispiel Ihre[`ShaderMaterial`](../shadermaterial) Instanz kann zwei Techniken haben, eine wird durch HLSL definiert und eine andere wird durch GLSL definiert. Unter einer Nicht-Fenster-Plattform sollte die GLSL anstelle von HLSL verwendet werden.

```csharp
public class ShaderMaterial : Material
```
