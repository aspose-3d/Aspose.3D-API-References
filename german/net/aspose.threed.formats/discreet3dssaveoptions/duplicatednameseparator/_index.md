## Discreet3dsSaveOptions.DuplicatedNameSeparator property

Das Trennzeichen zwischen dem Objektnamen und dem duplizierten Zähler, der Standardwert ist „_“. Wenn eine Szene Objekte mit demselben Namen enthält, generiert der Aspose.3D 3DS-Exporter einen anderen Namen für das Objekt. Zum Beispiel gibt es zwei Knoten mit dem Namen „Box“, der erste Knoten erhält den Namen „Box“, und der zweite Knoten erhält den neuen Namen „Box_2“, wobei die Standardkonfiguration verwendet wird.

public string DuplicatedNameSeparator { get; set; }
