## RvmSaveOptions.AttributePrefix property

Ermittelt oder setzt das Präfix, welche Attribute exportiert werden, die exportierte Eigenschaft enthält kein Präfix, benutzerdefinierte Eigenschaften mit unterschiedlichem Präfix werden nicht exportiert, der Standardwert ist 'rvm:'. Zum Beispiel, wenn eine Eigenschaft rvm:Refno ist =345, das exportierte Attribut ist Refno = 345, das Präfix wird entfernt.

public string AttributePrefix { get; set; }
