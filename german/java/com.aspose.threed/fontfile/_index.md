Die Font-Datei enthält Definitionen für Glyphen, die zur Erstellung eines Textprofils verwendet werden. **Example:** Der folgende Code zeigt, wie man ein 3D‑Mesh aus Text mit der angegebenen Font-Datei erstellt.

```
var font = FontFile.fromFile("CascadiaCode-Regular.otf");
             var text = new Text();
             text.setFont(font);
             text.setContent("ABC");
             text.setFontSize(10);
             var linear = new LinearExtrusion(text, 10).toMesh();
             var scene = new Scene(linear);
             scene.save("test.stl");
```
