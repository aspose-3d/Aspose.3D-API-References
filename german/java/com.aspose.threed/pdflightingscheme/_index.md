public enum PdfLightingScheme extends Enum<PdfLightingScheme>

LightingScheme gibt die Beleuchtung an, die auf 3D‑Grafiken angewendet wird.
## Felder

| Feld | Beschreibung |
| --- | --- |
| [ARTWORK](#ARTWORK) | Verwendet die in der Szene definierten Lichter. |
| [BLUE](#BLUE) | Drei blaue unendliche Lichter, kein Umgebungsanteil. |
| [CAD](#CAD) | Drei graue unendliche Lichter und ein Licht, das an die Kamera angehängt ist, kein Umgebungsanteil. |
| [CUBE](#CUBE) | Sechs graue unendliche Lichter, ausgerichtet an den Hauptachsen, kein Umgebungsanteil. |
| [DAY](#DAY) | Drei hellgraue unendliche Lichter, kein Umgebungsanteil. |
| [HARD](#HARD) | Drei graue unendliche Lichter, moderater Umgebungsanteil. |
| [HEADLAMP](#HEADLAMP) | Ein einzelnes unendliches Licht, an die Kamera angehängt, niedriger Umgebungsanteil. |
| [NIGHT](#NIGHT) | Ein gelbes, ein aqua und ein blaues unendliches Licht, kein Umgebungsanteil. |
| [NONE](#NONE) | Es werden keine Lichter verwendet. |
| [PRIMARY](#PRIMARY) | Ein rotes, ein grünes und ein blaues unendliches Licht, kein Umgebungsanteil. |
| [RED](#RED) | Drei rote unendliche Lichter, kein Umgebungsanteil |
| [WHITE](#WHITE) | Drei blau-graue unendliche Lichter, kein Umgebungsbegriff |
