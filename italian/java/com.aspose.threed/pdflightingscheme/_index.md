LightingScheme specifica l'illuminazione da applicare all'opera 3D.
## Campi

| Campo | Descrizione |
| --- | --- |
| [ARTWORK](#ARTWORK) | Utilizza le luci definite nella scena |
| [BLUE](#BLUE) | Tre luci infinite blu, senza termine ambientale |
| [CAD](#CAD) | Tre luci infinite grigie e una luce collegata alla fotocamera, senza termine ambientale |
| [CUBE](#CUBE) | Sei luci infinite grigie allineate con gli assi principali, senza termine ambientale |
| [DAY](#DAY) | Tre luci infinite grigio chiaro, senza termine ambientale |
| [HARD](#HARD) | Tre luci infinite grigie, termine ambientale moderato |
| [HEADLAMP](#HEADLAMP) | Una singola luce infinita collegata alla fotocamera, termine ambientale basso |
| [NIGHT](#NIGHT) | Una luce infinita gialla, una aqua e una blu, senza termine ambientale |
| [NONE](#NONE) | Nessuna luce è utilizzata. |
| [PRIMARY](#PRIMARY) | Una luce infinita rossa, una verde e una blu, senza termine ambientale |
| [RED](#RED) | Tre luci infinite rosse, senza termine ambientale |
| [WHITE](#WHITE) | Tre luci infinite blu-grigie, nessun termine ambientale |
