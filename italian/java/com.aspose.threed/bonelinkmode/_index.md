La modalità di collegamento di un osso si riferisce al modo in cui un osso è connesso o collegato al suo osso genitore all'interno di una struttura gerarchica.
## Campi

| Campo | Descrizione |
| --- | --- |
| [ADDITIVE](#ADDITIVE) | La modalità additiva calcola le trasformazioni delle ossa figlie aggiungendo le loro trasformazioni locali a quelle delle ossa genitore. |
| [NORMALIZE](#NORMALIZE) | In questa modalità, le trasformazioni delle ossa figlie sono normalizzate rispetto alle trasformazioni dell'osso genitore. |
| [TOTAL_ONE](#TOTAL-ONE) | Total One garantisce che le trasformazioni combinate delle ossa genitore e figlia producano una trasformazione combinata che si scala a una lunghezza complessiva di un'unità. |

### ADDITIVE {#ADDITIVE}

### NORMALIZE {#NORMALIZE}

In questa modalità, le trasformazioni delle ossa figlie sono normalizzate rispetto alle trasformazioni dell'osso genitore.

### TOTAL_ONE {#TOTAL-ONE}

Total One garantisce che le trasformazioni combinate delle ossa genitore e figlia producano una trasformazione combinata che si scala a una lunghezza complessiva di un'unità.
