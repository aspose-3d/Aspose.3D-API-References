## FVector3 class

| Nome | Descrizione |
| --- | --- |
| x | Il componente x. |
| y | Il componente y. |
| z | Il componente y. |
| ZERO | Il vettore Zero. |
| UNIT_SCALE | Il vettore di scala unitario con tutti i componenti pari a 1 |
