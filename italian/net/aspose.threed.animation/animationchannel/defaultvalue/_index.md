## AnimationChannel.DefaultValue property

Ottiene o imposta il valore Default del canale. Se un canale non ha sequenze di fotogrammi chiave collegate, il valore predefinito verrà utilizzato durante la valutazione dell'animazione. Uno scenario reale: l'animazione anima solo la coordinata x di un nodo, la y e la z sono non modificato, il valore predefinito verrà utilizzato durante la valutazione completa della traduzione.

public object DefaultValue { get; set; }
