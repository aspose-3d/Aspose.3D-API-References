## Bone class

Un osso definisce il sottoinsieme del punto di controllo della geometria e lo spessore di raccordo definito per ciascun punto di controllo. Il[`Bone`](../bone) l'oggetto non può essere utilizzato direttamente, a[`SkinDeformer`](../skindeformer) istanza viene utilizzata per deformare la geometria e[`SkinDeformer`](../skindeformer)viene fornito con un set di ossa, ogni osso collegato a un nodo. NOTA: un punto di controllo di una geometria può essere limitato a più di un osso.

```csharp
public class Bone : A3DObject
```
