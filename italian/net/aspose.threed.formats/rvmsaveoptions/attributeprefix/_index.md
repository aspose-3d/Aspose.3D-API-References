## RvmSaveOptions.AttributePrefix property

Ottiene o imposta il prefisso di quali attributi verranno esportati, la proprietà esportata non conterrà alcun prefisso, le proprietà personalizzate con prefisso diverso non verranno esportate, il valore predefinito è 'rvm:'. Ad esempio se una proprietà è rvm:Refno =345, l'attributo esportato sarà Refno = 345, il prefisso viene rimosso.

public string AttributePrefix { get; set; }
