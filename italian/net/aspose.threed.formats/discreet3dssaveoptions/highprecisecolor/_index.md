## Discreet3dsSaveOptions.HighPreciseColor property

Se questo è vero, il file 3ds generato utilizzerà un colore ad alta precisione, significa che ogni canale di rosso/verde/blu è in float a 32 bit. Altrimenti il file generato utilizzerà un colore a 24 bit, ogni canale utilizzerà 8 bit di byte. Il valore predefinito è falso, perché non tutte le applicazioni supportano il colore ad alta precisione.

public bool HighPreciseColor { get; set; }
