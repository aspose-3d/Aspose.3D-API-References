## FVector3 class

Un vector flotante con tres componentes.


## Propiedades

| Nombre | Descripción |
| --- | --- |
| x | El componente x. |
| y | El componente y. |
| z | El componente y. |
| CERO | El vector Cero. |
| UNIT_SCALE | El vector de escala unitaria con todos los componentes iguales a 1 |
