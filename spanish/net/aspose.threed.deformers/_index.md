## Clases

| Clase | Descripción |
| --- | --- |
| [Bone](./bone) | Un hueso define el subconjunto del punto de control de la geometría y el peso de mezcla definido para cada punto de control. El[`Bone`](../aspose.threed.deformers/bone) objeto no se puede utilizar directamente, un[`SkinDeformer`](../aspose.threed.deformers/skindeformer) instancia se utiliza para deformar la geometría, y[`SkinDeformer`](../aspose.threed.deformers/skindeformer)viene con un conjunto de huesos, cada hueso vinculado a un nodo. NOTA: Un punto de control de una geometría puede estar delimitado a más de un Hueso. |

| [MorphTargetChannel](./morphtargetchannel) | Un MorphTargetChannel es utilizado por[`MorphTargetDeformer`](../aspose.threed.deformers/morphtargetdeformer) para organizar las geometrías de destino. Algunos formatos de archivo como FBX admiten varios canales en paralelo. |
| [MorphTargetDeformer](./morphtargetdeformer) | MorphTargetDeformer proporciona animación por vértice. MorphTargetDeformer organiza todos los objetivos mediante[`MorphTargetChannel`](../aspose.threed.deformers/morphtargetchannel) , cada canal puede organizar múltiples objetivos. Un uso común del deformador de objetivo de morph es aplicar expresión facial a un personaje. Se pueden encontrar más detalles en https://en.wikipedia.org/wiki/Morph_target_animation |
| [SkinDeformer](./skindeformer) | Un deformador de piel contiene múltiples huesos para trabajar, cada hueso combina una parte de la geometría por los pesos del punto de control. |
