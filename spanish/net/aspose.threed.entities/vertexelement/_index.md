## VertexElement class

Clase base de elementos de vértice. Un tipo de elemento de vértice se identifica mediante VertexElementType. Un VertexElement describe cómo se asigna el elemento de vértice a una superficie geométrica y cómo se organiza la información de asignación en la memoria. Un VertexElement contiene Normales, UV u otro tipo de información.
