## ApertureMode enumeration

Modos de apertura de la cámara. El modo de apertura determina qué valores controlan la apertura de la cámara. Si el modo de apertura es HorizAndVert, Horizontal o Vertical, se utiliza el campo de visión. Si el modo de apertura es FocalLength, entonces se usa la distancia focal.

```csharp
public enum ApertureMode
```

### Valores

| Nombre | Valor | Descripción |
| --- | --- | --- |
| HorizAndVert | `0` | Configure los valores de ángulo tanto para la configuración horizontal como para la vertical. |
| Horizontal | `1` | Establece solo el ángulo horizontal. |
| Vertical | `2` | Establece solo el ángulo vertical. |
| FocalLength | `3` | Usar distancia focal directamente. |
