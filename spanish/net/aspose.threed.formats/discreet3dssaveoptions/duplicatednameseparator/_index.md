## Discreet3dsSaveOptions.DuplicatedNameSeparator property

El separador entre el nombre del objeto y el contador duplicado, el valor predeterminado es "_". Cuando la escena contiene objetos que usan el mismo nombre, el exportador 3DS de Aspose.3D generará un nombre diferente para el objeto. Por ejemplo, hay dos nodos llamado "Box", el primer nodo tendrá un nombre "Box", y el segundo nodo obtendrá un nuevo nombre "Box_2" utilizando la configuración predeterminada.

public string DuplicatedNameSeparator { get; set; }
