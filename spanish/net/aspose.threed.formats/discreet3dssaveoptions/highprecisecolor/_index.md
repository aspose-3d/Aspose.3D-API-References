## Discreet3dsSaveOptions.HighPreciseColor property

Si esto es cierto, el archivo 3ds generado usará un color de alta precisión, lo que significa que cada canal rojo/verde/azul está en 32 bits flotantes. De lo contrario, el archivo generado usará un color de 24 bits, cada canal usará un byte de 8 bits. El valor predeterminado es falso, porque no todas las aplicaciones admiten el color de alta precisión.
