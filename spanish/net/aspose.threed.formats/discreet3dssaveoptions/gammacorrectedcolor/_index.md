## Discreet3dsSaveOptions.GammaCorrectedColor property

Un archivo 3ds puede contener el color original y el color con corrección de gamma para el mismo atributo, Si se establece en verdadero, se usará el color con corrección de gamma si es posible, de lo contrario, Aspose.3D intentará usar el color original.
