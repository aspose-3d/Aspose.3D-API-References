## RvmSaveOptions.AttributePrefix property

Obtiene o establece el prefijo de los atributos que se exportarán, la propiedad exportada no tendrá prefijo, las propiedades personalizadas con prefijo diferente no se exportarán, el valor predeterminado es 'rvm:'. Por ejemplo, si una propiedad es rvm:Refno =345, el atributo exportado será Refno = 345, el prefijo se elimina.

public string AttributePrefix { get; set; }
