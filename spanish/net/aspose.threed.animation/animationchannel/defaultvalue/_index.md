## AnimationChannel.DefaultValue property

Obtiene o establece el valor predeterminado del canal. Si un canal no tiene secuencias de fotogramas clave conectadas, el valor predeterminado se utilizará durante la evaluación de la animación. Un escenario real: la animación solo anima la coordenada x de un nodo, la y y la z son no cambiado, entonces se usará el valor predeterminado durante la evaluación de traducción completa.

public object DefaultValue { get; set; }
