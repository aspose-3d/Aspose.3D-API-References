## Clases

| Clase | Descripción |

| [AnimationChannel](./animationchannel) | Un canal asigna el campo de componente de la propiedad a un conjunto de secuencias de fotogramas clave |

| [AnimationNode](./animationnode) | Aspose.3D admite la jerarquía de animación, cada animación puede estar compuesta por varias animaciones y la definición de fotogramas clave de la animación. [`AnimationNode`](../aspose.threed.animation/animationnode) define la transformación de un valor de propiedad a lo largo del tiempo, por ejemplo, el nodo de animación se puede usar para controlar la transformación de un nodo u otro[`A3DObject`](../aspose.threed/a3dobject) propiedades numéricas del objeto. |
| [BindPoint](./bindpoint) | A[`BindPoint`](../aspose.threed.animation/bindpoint)generalmente se crea en la propiedad de un objeto, algunos tipos de propiedad contienen múltiples campos de componentes (como un campo Vector3), [`BindPoint`](../aspose.threed.animation/bindpoint) generará un canal para cada campo componente y conectará el campo a una o más instancias de secuencia de fotogramas clave a través de los canales. |
| [Extrapolation](./extrapolation) | La extrapolación define cómo hacerlo cuando el valor muestreado está fuera del rango definido por el primer y el último fotograma clave. |
| [KeyFrame](./keyframe) | Un cuadro clave se define principalmente por un tiempo y un valor, para algunos tipos de interpolación, la tangente/tensión/sesgo/continuidad también se usa para calcular el valor final de la muestra. Los valores muestreados en una posición de tiempo que no es un cuadro clave se interpolan por fotogramas clave entre los fotogramas clave anterior y siguiente Los valores antes/después del primer/último fotograma clave son calculados por el[`Extrapolation`](../aspose.threed.animation/extrapolation) clase. |
| [KeyframeSequence](./keyframesequence) | La secuencia de fotogramas clave, describe la transformación de un valor muestreado a lo largo del tiempo. |
