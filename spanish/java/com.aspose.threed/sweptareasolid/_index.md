Un [SweptAreaSolid](../../com.aspose.threed/sweptareasolid) construye una geometría al barrer un perfil a lo largo de una directriz. **Example:** El siguiente código muestra cómo modelar una entidad sólida al barrer una forma en C sobre un círculo

```
var directrix = new Circle(20);
         var shape = new CShape();
 
         var swept = new SweptAreaSolid();
         swept.setShape(shape);
         swept.setDirectrix(directrix);
         swept.setStartPoint(EndPoint.fromDegree(0));
         swept.setEndPoint(EndPoint.fromDegree(130));
 
         var scene = new Scene();
         scene.getRootNode().createChildNode(swept);
         scene.save("swept.obj");
```
