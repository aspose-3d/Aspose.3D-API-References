LightingScheme especifica la iluminación a aplicar al arte 3D.
## Campos

| Campo | Descripción |
| --- | --- |
| [ARTWORK](#ARTWORK) | Utiliza las luces definidas en la escena |
| [BLUE](#BLUE) | Tres luces infinitas azules, sin término ambiental |
| [CAD](#CAD) | Tres luces infinitas grises y una luz adjunta a la cámara, sin término ambiental |
| [CUBE](#CUBE) | Seis luces infinitas grises alineadas con los ejes principales, sin término ambiental |
| [DAY](#DAY) | Tres luces infinitas gris claro, sin término ambiental |
| [HARD](#HARD) | Tres luces infinitas grises, término ambiental moderado |
| [HEADLAMP](#HEADLAMP) | Una única luz infinita adjunta a la cámara, término ambiental bajo |
| [NIGHT](#NIGHT) | Una luz infinita amarilla, una aqua y una azul, sin término ambiental |
| [NONE](#NONE) | No se utilizan luces. |
| [PRIMARY](#PRIMARY) | Una luz infinita roja, una verde y una azul, sin término ambiental |
| [RED](#RED) | Tres luces infinitas rojas, sin término ambiental |
| [WHITE](#WHITE) | Tres luces azul‑gris infinitas, sin término ambiental |
