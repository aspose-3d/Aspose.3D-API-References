```
public class TransformedCurve extends Curve
```

Un [TransformedCurve](../../com.aspose.threed/transformedcurve) da una posición a una curva mediante una matriz de transformación. Esto permite realizar una transformación dentro de un [TrimmedCurve](../../com.aspose.threed/trimmedcurve) o un [CompositeCurve](../../com.aspose.threed/compositecurve).
