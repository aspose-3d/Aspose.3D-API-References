Файл шрифта содержит определения глифов, он используется для создания текстового профиля. **Example:** Следующий код показывает, как создать 3D‑модель из текста, используя указанный файл шрифта.

```
var font = FontFile.fromFile("CascadiaCode-Regular.otf");
             var text = new Text();
             text.setFont(font);
             text.setContent("ABC");
             text.setFontSize(10);
             var linear = new LinearExtrusion(text, 10).toMesh();
             var scene = new Scene(linear);
             scene.save("test.stl");
```
