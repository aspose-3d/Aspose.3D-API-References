LightingScheme указывает освещение, применяемое к 3D‑артворку.
## Поля

| Поле | Описание |
| --- | --- |
| [ARTWORK](#ARTWORK) | Использует источники света, определённые в сцене |
| [BLUE](#BLUE) | Три синих бесконечных источника света, без фонового освещения |
| [CAD](#CAD) | Три серых бесконечных источника света и один свет, привязанный к камере, без фонового освещения |
| [CUBE](#CUBE) | Шесть серых бесконечных источников света, выровненных по основным осям, без фонового освещения |
| [DAY](#DAY) | Три светло-серых бесконечных источника света, без фонового освещения |
| [HARD](#HARD) | Три серых бесконечных источника света, умеренный фоновый свет |
| [HEADLAMP](#HEADLAMP) | Один бесконечный свет, привязанный к камере, низкий фоновый свет |
| [NIGHT](#NIGHT) | Один желтый, один аква и один синий бесконечный свет, без фонового освещения |
| [NONE](#NONE) | Свет не используется. |
| [PRIMARY](#PRIMARY) | Один красный, один зеленый и один синий бесконечный свет, без фонового освещения |
| [RED](#RED) | Три красных бесконечных источника света, без фонового освещения |
| [WHITE](#WHITE) | Три синевато-серых бесконечных света, без фонового терма |
