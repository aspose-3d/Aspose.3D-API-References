Профиль текста, этот профиль описывает контуры, используя шрифт и текст. **Пример:** Следующий код показывает, как создать 3D-сетку из текста, используя указанный файл шрифта.

```
var font = FontFile.fromFile("CascadiaCode-Regular.otf");
             var text = new Text();
             text.setFont(font);
             text.setContent("ABC");
             text.setFontSize(10);
             var linear = new LinearExtrusion(text, 10).toMesh();
             var scene = new Scene(linear);
             scene.save("test.stl");
```
