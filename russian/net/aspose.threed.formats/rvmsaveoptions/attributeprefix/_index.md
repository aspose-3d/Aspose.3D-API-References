## RvmSaveOptions.AttributePrefix property

Получает или задает префикс атрибутов, которые будут экспортированы, экспортированное свойство не будет содержать префикс, пользовательские свойства с другим префиксом не будут экспортированы, значение по умолчанию ' рвм:'. Например, если свойство имеет значение rvm:Refno=345, экспортируемый атрибут будет иметь значение Refno = 345, префикс удаляется.

```csharp
public string AttributePrefix { get; set; }
```
