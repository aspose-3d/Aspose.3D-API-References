## RvmSaveOptions.AttributePrefix property

Получает или задает префикс экспортируемых атрибутов, экспортируемое свойство не будет содержать префикса, пользовательские свойства с другим префиксом не будут экспортированы, значение по умолчанию — 'rvm:'. Например, если свойство rvm:Refno =345, экспортируемый атрибут будет Refno = 345, префикс удален.

public string AttributePrefix { get; set; }
