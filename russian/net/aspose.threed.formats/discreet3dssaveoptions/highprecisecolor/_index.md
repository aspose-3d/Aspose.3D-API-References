## Discreet3dsSaveOptions.HighPreciseColor property

Если это правда, сгенерированный файл 3ds будет использовать высокоточный цвет, что означает, что каждый канал красного/зеленого/синего находится в 32-битном плавающем значении. В противном случае сгенерированный файл будет использовать 24-битный цвет, каждый канал использует 8-битный байт. Значение по умолчанию - false, так как не все приложения поддерживают высокоточный цвет.

public bool HighPreciseColor { get; set; }
