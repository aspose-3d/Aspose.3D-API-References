## Discreet3dsSaveOptions.HighPreciseColor property

Если это так, сгенерированный файл 3ds будет использовать высокоточный цвет, что означает, что каждый канал красного/зеленого/синего цвета имеет 32-битное число с плавающей запятой. В противном случае сгенерированный файл будет использовать 24-битный цвет, каждый канал использует 8-битный байт. Значение по умолчанию неверно, потому что не все приложения поддерживают высокоточный цвет.

public bool HighPreciseColor { get; set; }
