## Discreet3dsSaveOptions.DuplicatedNameSeparator property

Разделитель между именем объекта и повторяющимся счетчиком, значение по умолчанию "_". Если сцена содержит объекты с одинаковыми именами, экспортер Aspose.3D 3DS сгенерирует другое имя для объекта. Например, есть два узла с именем "Box", первый узел будет иметь имя "Box", , а второй узел получит новое имя "Box_2" с использованием конфигурации по умолчанию.

public string DuplicatedNameSeparator { get; set; }
