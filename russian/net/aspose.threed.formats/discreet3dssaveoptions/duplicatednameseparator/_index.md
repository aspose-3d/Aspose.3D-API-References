## Discreet3dsSaveOptions.DuplicatedNameSeparator property

Разделитель между именем объекта и повторяющимся счетчиком, значение по умолчанию "_". Когда сцена содержит объекты с одинаковыми именами, экспортер Aspose.3D 3DS создаст другое имя для объекта. Например, есть два узла с именем "Box", первый узел будет иметь имя "Box", а второй узел получит новое имя "Box_2", используя значение по умолчанию. конфигурация.
