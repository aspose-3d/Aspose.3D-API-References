## TriMesh.FromRawData method

Создать TriMesh из необработанных данных

```csharp
public static TriMesh FromRawData(VertexDeclaration vd, byte[] vertices, int[] indices, 
    bool generateVertexMapping)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| vd | VertexDeclaration | Объявление вершины должно содержать хотя бы одно поле. |
| vertices | Byte[] | Входные данные вершины, минимальная длина вершин должна быть больше или равна размеру объявления вершины. |
| indices | Int32[] | Индексы треугольника |
| generateVertexMapping | Boolean | Создать[`Vertex`](../../../aspose.threed.utilities/vertex) для каждой вершины, что не обязательно только для сериализации/десериализации. |

### Возвращаемое значение

[`TriMesh`](../../trimesh) экземпляр, инкапсулирующий массив входных байтов.

### Примечания

Возвращенный TriMesh не будет копировать входной массив байтов для производительности, внешние изменения в массиве будут отражены в этом экземпляре.
