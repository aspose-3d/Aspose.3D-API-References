## ApertureMode enumeration

Режимы апертуры камеры. Режим апертуры определяет, какие значения управляют апертурой камеры. Если установлен режим апертуры HorizAndVert, Horizontal или Vertical, то используется поле зрения. Если режим диафрагмы FocalLength, то используется фокусное расстояние.

```csharp
public enum ApertureMode
```

### Ценности

| Имя | Ценность | Описание |
| --- | --- | --- |
| HorizAndVert | `0` | Установите значения угла как для горизонтального, так и для вертикального параметров. |
| Horizontal | `1` | Установить только горизонтальный угол. |
| Vertical | `2` | Установить только вертикальный угол. |
| FocalLength | `3` | Использовать фокусное расстояние напрямую. |
