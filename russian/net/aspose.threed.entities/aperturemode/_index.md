## ApertureMode enumeration

Режимы диафрагмы камеры. Режим апертуры определяет, какие значения управляют апертурой камеры. Если установлен режим апертуры HorizAndVert, Horizontal или Vertical, то используется поле зрения. Если установлен режим диафрагмы FocalLength, то используется фокусное расстояние.

```csharp
public enum ApertureMode
```

### Ценности

| Имя | Ценность | Описание |
| --- | --- | --- |
| HorizAndVert | `0` | Установите значения угла как для горизонтального, так и для вертикального параметров. |
| Horizontal | `1` | Установите только горизонтальный угол. |
| Vertical | `2` | Установите только вертикальный угол. |
| FocalLength | `3` | Использовать фокусное расстояние напрямую. |
