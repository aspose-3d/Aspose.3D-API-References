Aspose.3D предоставляет модель затенения для[`LambertMaterial`](../aspose.threed.shading/lambertmaterial) ,[`PhongMaterial`](../aspose.threed.shading/phongmaterial) а также[`ShaderMaterial`](../aspose.threed.shading/shadermaterial) |

| [Texture](./texture) | Этот класс определяет текстуру из внешнего файла. |
| [TextureBase](./texturebase) | Базовый класс для всех текстур бетона. Текстура определяет внешний вид геометрической поверхности. |
