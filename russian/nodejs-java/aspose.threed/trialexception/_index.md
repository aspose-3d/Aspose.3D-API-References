## TrialException class

Это исключение возникает в Scene.Open/Scene.Save, когда лицензии не применены.  Вы можете отключить это исключение, установив SuppressTrialException в true.

### getSuppressTrialException{#getSuppressTrialException}

| Имя | Описание |
| --- | --- |
| getSuppressTrialException() | Устанавливает значение true, чтобы подавить исключение пробной версии при использовании без лицензии, но ограничения не будут сняты. Чтобы снять ограничения, пожалуйста, используйте правильную лицензию. Установка значения true также означает, что вы осведомлены о ограничениях при использовании без лицензии. |
