## VertexElement class

Базовый класс элементов вершины. Тип элемента вершины определяется VertexElementType. VertexElement описывает, как элемент вершины отображается на поверхность геометрии и как информация о отображении размещается в памяти. VertexElement содержит нормали, UV или другую информацию.  @hideconstructor
