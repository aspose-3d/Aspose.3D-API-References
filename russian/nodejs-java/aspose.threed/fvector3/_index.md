## FVector3 class

| --- | --- |
| x | Компонент x. |
| y | Компонент y. |
| z | Компонент y. |
| НОЛЬ | Нулевой вектор. |
| UNIT_SCALE | Вектор единичного масштаба, у которого все компоненты равны 1 |
