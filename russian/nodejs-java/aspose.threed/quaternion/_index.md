## Quaternion class

Кватернион обычно используется для выполнения вращения в компьютерной графике.

| --- | --- |
| w | Компонент w. |
| x | Компонент x. |
| y | Компонент y. |
| z | Компонент z. |
| IDENTITY | Единичный кватернион. |
