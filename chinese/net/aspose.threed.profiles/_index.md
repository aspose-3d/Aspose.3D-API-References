所有 2D 剖面类都定义在此命名空间中。

## 类

| 类 | 描述 |
| --- | --- |
| [ArbitraryProfile](./arbitraryprofile/) | 此类允许您直接从任意曲线构建 2D 配置文件。 |
| [CenterLineProfile](./centerlineprofile/) | 兼容 IFC 的中心线配置文件 |
| [CircleShape](./circleshape/) | 兼容 IFC 的圆形配置文件，可通过 [`LinearExtrusion`](../aspose.threed.entities/linearextrusion/) 构建网格。 |
| [CShape](./cshape/) | 兼容 IFC 的 C 形配置文件，由参数定义。配置文件的中心位置位于包围盒的中心。 |
| [EllipseShape](./ellipseshape/) | 兼容 IFC 的椭圆形，由参数定义。配置文件的中心位置位于包围盒的中心。 |
| [FontFile](./fontfile/) | 字体文件包含字形定义，用于创建文本配置文件。 |
| [HollowCircleShape](./hollowcircleshape/) | 兼容 IFC 的空心圆形配置文件。 |
| [HollowRectangleShape](./hollowrectangleshape/) | 兼容 IFC 的空心矩形形状，具有内外圆角。 |
| [HShape](./hshape/) | [`HShape`](../aspose.threed.profiles/hshape/) 提供了 'H' 或 'I' 形状的定义参数。 |
| [LShape](./lshape/) | 兼容 IFC 的 L 形配置文件，由参数定义。 |
| [MirroredProfile](./mirroredprofile/) | 兼容 IFC 的镜像配置文件。该配置文件通过沿 y 轴镜像基准配置文件来定义新配置文件。 |
| [ParameterizedProfile](./parameterizedprofile/) | 所有参数化配置文件的基类。 |
| [Profile](./profile/) | xy 平面中的 2D 配置文件 |
| [RectangleShape](./rectangleshape/) | 兼容 IFC 的带圆角矩形形状。 |
| [Text](./text/) | 文本配置文件，该配置文件使用字体和文本描述轮廓。 |
| [TrapeziumShape](./trapeziumshape/) | 兼容 IFC 的梯形形状，由参数定义。 |
| [TShape](./tshape/) | 兼容 IFC 的 T 形，由参数定义。 |
| [UShape](./ushape/) | 兼容 IFC 的 U 形，由参数定义。 |
| [ZShape](./zshape/) | 兼容 IFC 的 Z 形配置文件，由参数定义。 |
