## GltfSaveOptions.ImageFormat property

标准 glTF 仅支持 PNG/JPG 作为纹理格式，此选项将指导 Aspose.3D 在导出期间如何将非标准图像转换为支持的格式。默认值为 Png。

public GltfEmbeddedImageFormat ImageFormat { get; set; }
