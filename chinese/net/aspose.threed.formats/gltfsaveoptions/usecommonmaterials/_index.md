## GltfSaveOptions.UseCommonMaterials property

使用 KHR 通用材质扩展序列化材质，默认值为 false。将此设置为 false 将导致 Aspose.3D 在 ExportShaders 时导出一套顶点/片段着色器。

```csharp
public bool UseCommonMaterials { get; set; }
```

## 备注

此属性仅适用于 glTF 1.0
