## Discreet3dsLoadOptions.GammaCorrectedColor property

3ds 文件可能包含同一属性的原始颜色和伽马校正颜色，将此设置为 true 将在可能的情况下使用伽马校正颜色，否则 Aspose.3D 将尝试使用原始颜色。

```csharp
public bool GammaCorrectedColor { get; set; }
```
