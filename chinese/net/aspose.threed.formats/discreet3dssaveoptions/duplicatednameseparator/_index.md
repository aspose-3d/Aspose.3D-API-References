## Discreet3dsSaveOptions.DuplicatedNameSeparator property

对象名称与重复计数器之间的分隔符，默认值为 "_"。当场景中包含使用相同名称的对象时，Aspose.3D 3DS 导出器将为该对象生成不同的名称。例如，有两个名为 "Box" 的节点，第一个节点的名称为 "Box"，第二个节点将在默认配置下获得新名称 "Box_2"。

```csharp
public string DuplicatedNameSeparator { get; set; }
```
