## KeyFrame.StepMode property

获取或设置关键帧的步进模式。如果插值类型为 Constant，list.data[index] 决定在插值期间使用哪个关键帧的值。PreviousValue 表示使用左侧关键帧的值，NextValue 表示使用下一个右侧关键帧的值。

```csharp
public StepMode StepMode { get; set; }
```

### Property Value

步进模式。
