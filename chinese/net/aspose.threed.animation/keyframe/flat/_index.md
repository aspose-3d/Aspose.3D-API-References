## KeyFrame.Flat property

获取或设置关键帧是否为平坦。如果下一个或上一个关键帧具有相同的值，则关键帧应为平坦。平坦的关键帧具有平坦的切线和固定的插值。

```csharp
public bool Flat { get; set; }
```
