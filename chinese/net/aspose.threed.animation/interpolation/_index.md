## Interpolation enumeration

关键帧的插值类型。

```csharp
public enum Interpolation
```

### 值

| 名称 | 值 | 描述 |
| --- | --- | --- |
| Constant | `0` | 该值将在下一个段之前保持与第一个点的值相同。 |
| Linear | `1` | 线性插值是两点之间的直线。 |
| Bezier | `2` | 贝塞尔或 Hermite 样条。 |
| BSpline | `3` | 基样条由一系列控制点定义，曲线仅保证经过第一个和最后一个点。 |
| CardinalSpline | `4` | 基数样条是一种三次 Hermite 样条，其切线由端点和张力参数定义。 |
| TCBSpline | `5` | 亦称 Kochanek-Bartels 样条，切线的行为由张力/偏差/连续性定义 |
