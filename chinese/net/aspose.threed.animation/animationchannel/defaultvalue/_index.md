## AnimationChannel.DefaultValue property

获取或设置通道的默认值。如果通道没有连接任何关键帧序列，则在动画评估期间将使用默认值。实际场景：动画仅对节点的 x 坐标进行动画，而 y 和 z 坐标不变，则在完整平移评估期间将使用默认值。

```csharp
public object DefaultValue { get; set; }
```
