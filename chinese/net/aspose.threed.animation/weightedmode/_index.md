## WeightedMode enumeration

加权模式。

```csharp
[Flags]
public enum WeightedMode
```

### 值

| 名称 | 值 | 描述 |
| --- | --- | --- |
| None | `0` | 出切线和下一个入切线的权重均不使用。当计算需要切线信息时，将使用默认值（0.3333）。 |
| OutWeight | `1` | 出（右）切线已加权。 |
| NextInWeight | `2` | 下一个入（左）切线已加权。 |
| Both | `3` | 出切线和下一个入切线均已加权。 |
