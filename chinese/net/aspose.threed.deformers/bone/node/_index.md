## Bone.Node property

获取或设置节点。骨骼节点是皮肤附着的骨骼，[`SkinDeformer`](../../skindeformer/) 将使用该骨骼节点影响控制点的位移。骨骼节点通常附带一个 [`Skeleton`](../../../aspose.threed.entities/skeleton/)，但并非必需。附带的 [`Skeleton`](../../../aspose.threed.entities/skeleton/) 通常被 DCC 软件用于向用户显示骨架。

```csharp
public Node Node { get; set; }
```
