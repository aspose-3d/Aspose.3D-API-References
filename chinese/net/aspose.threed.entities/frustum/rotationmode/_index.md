## Frustum.RotationMode property

获取或设置视锥的方向模式。此属性仅在 [`Target`](../target/) 为 null 时有效。如果值为 FixedTarget，方向始终由属性 [`LookAt`](../lookat/) 计算；否则 [`LookAt`](../lookat/) 始终由 [`Direction`](../direction/) 计算。

public RotationMode RotationMode { get; set; }
