## Shape class

形状描述了一组控制点上的变形，类似于 Maya 中的簇变形器。 例如，我们可以将形状添加到创建的几何图形中。 形状和几何具有相同的拓扑信息，但控制点的位置不同。 几何体具有不同程度的影响，会产生变形效果。

```csharp
public class Shape : Geometry
```
