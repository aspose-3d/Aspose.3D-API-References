## ApertureMode enumeration

相机光圈模式。光圈模式决定哪些数值驱动相机光圈。如果光圈模式为 HorizAndVert、Horizontal 或 Vertical，则使用视场角。如果光圈模式为 FocalLength，则使用焦距。

```csharp
public enum ApertureMode
```

### 值

| 名称 | 值 | 描述 |
| --- | --- | --- |
| HorizAndVert | `0` | 设置水平和垂直设置的角度值。 |
| Horizontal | `1` | 仅设置水平角度。 |
| Vertical | `2` | 仅设置垂直角度。 |
| FocalLength | `3` | 直接使用焦距。 |
