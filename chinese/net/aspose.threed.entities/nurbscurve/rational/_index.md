## NurbsCurve.Rational property

获取或设置它是否为有理，此值指示此 [`NurbsCurve`](../) 是有理样条还是无理样条。无理 B-样条是有理 B-样条的特例。

```csharp
public bool Rational { get; set; }
```

### Property Value

如果是有理样条则为 true；否则为 false，表示无理样条。
