## 课程

| 班级 | 描述 |
| --- | --- |
| [LambertMaterial](./lambertmaterial) | 朗伯着色模型材质 |
| [Material](./material) | 材质定义几何视觉外观所需的参数。 Aspose.3D 为[`LambertMaterial`](../aspose.threed.shading/lambertmaterial),[`PhongMaterial`](../aspose.threed.shading/phongmaterial)和[`ShaderMaterial`](../aspose.threed.shading/shadermaterial) |
| [PbrMaterial](./pbrmaterial) | 基于反照率颜色/金属/粗糙度的基于物理的渲染材质 |
| [PbrSpecularMaterial](./pbrspecularmaterial) | 基于漫反射颜色/镜面反射/光泽度的基于物理的渲染材质 |
| [PhongMaterial](./phongmaterial) | blinn-phong 着色模型的材质。 |
| [ShaderMaterial](./shadermaterial) | 着色器材质允许通过外部渲染引擎或着色器语言来描述材质。 [`ShaderMaterial`](../aspose.threed.shading/shadermaterial)用途[`ShaderTechnique`](../aspose.threed.shading/shadertechnique)描述具体的渲染细节， 会根据最终的渲染平台使用最合适的 比如你的[`ShaderMaterial`](../aspose.threed.shading/shadermaterial)instance可以有两种技术，一种由HLSL定义，另一种由GLSL 定义_在非window平台下，应该使用GLSL而不是HLSL |
| [ShaderTechnique](./shadertechnique) | 着色器技术代表具体的渲染实现。 |
| [Texture](./texture) | 此类定义来自外部文件的纹理。 |
| [TextureBase](./texturebase) | 所有混凝土纹理的基类。 纹理定义几何表面的外观。 |
| [TextureSlot](./textureslot) | 中的纹理槽[`Material`](../aspose.threed.shading/material) 可以通过材质实例枚举. |
