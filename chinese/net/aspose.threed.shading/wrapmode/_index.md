## WrapMode enumeration

```csharp
public enum WrapMode
```

### 值

| 名称 | 值 | 描述 |
| --- | --- | --- |
| Wrap | `0` | 在模型表面平铺纹理，创建重复图案。 |
| Clamp | `1` | 在边界将纹理夹紧到最后一个像素。 |
| Mirror | `2` | 纹理将被重复，但当坐标的整数部分为奇数时会进行镜像。 |
| MirrorOnce | `3` | 纹理将被镜像一次，然后夹紧到最大值。 |
| Border | `4` | 超出范围 [0.0, 1.0] 的坐标将设置为指定的边框颜色。 |
