## VertexElement class

vertex 元素的基类。  vertex 元素类型由 VertexElementType 标识。  VertexElement 描述 vertex 元素如何映射到几何表面以及映射信息在内存中的排列方式。  VertexElement 包含 Normals、UVs 或其他类型的信息。  @hideconstructor
