## VertexElement class

الفئة الأساسية لعناصر الرأس. يتم تحديد نوع عنصر الرأس بواسطة VertexElementType. يصف VertexElement كيفية ربط عنصر الرأس بسطح الهندسة وكيفية ترتيب معلومات الربط في الذاكرة. يحتوي VertexElement على Normals و UVs أو أي نوع آخر من المعلومات.  @hideconstructor
