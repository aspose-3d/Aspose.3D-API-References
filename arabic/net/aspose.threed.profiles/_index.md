جميع فئات الملف الشخصي ثنائي الأبعاد معرفة في هذه المساحة.

## الفئات

| الفئة | الوصف |
| --- | --- |
| [ArbitraryProfile](./arbitraryprofile/) | هذه الفئة تسمح لك بإنشاء ملف شخصي ثنائي الأبعاد مباشرةً من منحنى عشوائي. |
| [CenterLineProfile](./centerlineprofile/) | ملف شخصي خط مركزي متوافق مع IFC |
| [CircleShape](./circleshape/) | ملف شخصي دائرة متوافق مع IFC، يمكن استخدامه لإنشاء شبكة عبر [`LinearExtrusion`](../aspose.threed.entities/linearextrusion/) |
| [CShape](./cshape/) | ملف شخصي شكل C متوافق مع IFC يُحدد بالمعلمات. موضع المركز للملف الشخصي في وسط صندوق الحدود. |
| [EllipseShape](./ellipseshape/) | شكل بيضاوي متوافق مع IFC يُحدد بالمعلمات. موضع المركز للملف الشخصي في وسط صندوق الحدود. |
| [FontFile](./fontfile/) | ملف الخط يحتوي على تعريفات للرموز، يُستخدم لإنشاء ملف شخصي نصي. |
| [HollowCircleShape](./hollowcircleshape/) | ملف شخصي دائرة مجوفة متوافق مع IFC. |
| [HollowRectangleShape](./hollowrectangleshape/) | شكل مستطيل مجوف متوافق مع IFC مع زوايا داخلية/خارجية مستديرة. |
| [HShape](./hshape/) | الـ[`HShape`](../aspose.threed.profiles/hshape/) يوفر المعلمات التعريفية لشكل 'H' أو 'I'. |
| [LShape](./lshape/) | ملف شخصي شكل L متوافق مع IFC يُحدد بالمعلمات. |
| [MirroredProfile](./mirroredprofile/) | ملف شخصي مرآة متوافق مع IFC. هذا الملف يحدد ملفًا جديدًا عن طريق عكس الملف الأساسي حول محور y. |
| [ParameterizedProfile](./parameterizedprofile/) | الفئة الأساسية لجميع الملفات الشخصية المعيّنة بالمعلمات. |
| [Profile](./profile/) | ملف شخصي ثنائي الأبعاد في مستوى xy |
| [RectangleShape](./rectangleshape/) | شكل مستطيل متوافق مع IFC مع زوايا مستديرة. |
| [Text](./text/) | ملف شخصي نصي، يصف هذا الملف الشخصي الخطوط باستخدام الخط والنص. |
| [TrapeziumShape](./trapeziumshape/) | شكل شبه منحرف متوافق مع IFC يُحدد بالمعلمات. |
| [TShape](./tshape/) | شكل T متوافق مع IFC يُحدد بالمعلمات. |
| [UShape](./ushape/) | شكل U متوافق مع IFC يُحدد بالمعلمات. |
| [ZShape](./zshape/) | ملف شخصي شكل Z متوافق مع IFC يُحدد بالمعلمات. |
