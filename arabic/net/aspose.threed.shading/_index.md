جميع الفئات المتعلقة بالتظليل معرفة في هذه المساحة.

## الفئات

| الفئة | الوصف |
| --- | --- |
| [LambertMaterial](./lambertmaterial/) | المادة لنموذج التظليل لامبرت |
| [Material](./material/) | المادة تعرف المعلمات اللازمة للمظهر البصري للهندسة. توفر Aspose.3D نموذج تظليل لـ [`LambertMaterial`](../aspose.threed.shading/lambertmaterial/), [`PhongMaterial`](../aspose.threed.shading/phongmaterial/) و [`ShaderMaterial`](../aspose.threed.shading/shadermaterial/) |
| [PbrMaterial](./pbrmaterial/) | المادة للتصيير الفيزيائي القائم على لون الألبيدو/المعدن/الخشونة |
| [PbrSpecularMaterial](./pbrspecularmaterial/) | المادة للتصيير الفيزيائي القائم على اللون المنتشر/اللمعان/اللمعان |
| [PhongMaterial](./phongmaterial/) | المادة لنموذج التظليل بلين-فونغ. |
| [ShaderMaterial](./shadermaterial/) | تسمح مادة التظليل بوصف المادة بواسطة محرك تصيير خارجي أو لغة تظليل. يستخدم [`ShaderMaterial`](../aspose.threed.shading/shadermaterial/) [`ShaderTechnique`](../aspose.threed.shading/shadertechnique/) لوصف تفاصيل التصيير المحددة، وسيتم اختيار الأنسب وفقًا لمنصة التصيير النهائية. على سبيل المثال، يمكن أن يحتوي كائن [`ShaderMaterial`](../aspose.threed.shading/shadermaterial/) الخاص بك على تقنيتين، إحداهما معرفة بـ HLSL، والأخرى معرفة بـ GLSL. تحت المنصات غير النافذة يجب استخدام GLSL بدلاً من HLSL |
| [ShaderTechnique](./shadertechnique/) | تقنية التظليل تمثل تنفيذًا محددًا للتصيير. |
| [Texture](./texture/) | هذه الفئة تعرف الملمس من ملف خارجي. |
| [TextureBase](./texturebase/) | الفئة الأساسية لجميع الملمسات المحددة. الملمس يحدد الشكل والمظهر لسطح الهندسة. |
| [TextureSlot](./textureslot/) | فتحة الملمس في [`Material`](../aspose.threed.shading/material/), يمكن تعدادها عبر كائن المادة. |
## التعداد

| التعداد | الوصف |
| --- | --- |
| [AlphaSource](./alphasource/) | يحدد ما إذا كان الملمس يحتوي على قناة ألفا. |
| [TextureFilter](./texturefilter/) | خيارات الفلترة أثناء أخذ عينات الملمس. |
| [WrapMode](./wrapmode/) | وضع التغليف للملمس. |
