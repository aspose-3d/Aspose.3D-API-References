## TrialException.SuppressTrialException property

يضبط هذا إلى true لكتم استثناء النسخة التجريبية عند الاستخدام غير المرخص، لكن القيود لن تُرفع. لرفع القيود، يرجى استخدام ترخيص صحيح. وضبط هذا إلى true يعني أيضًا أنك على علم بالقيود غير المرخصة.

```csharp
public static bool SuppressTrialException { get; set; }
```
