## GltfSaveOptions.UseCommonMaterials property

تسلسل المواد باستخدام امتدادات مادة KHR الشائعة، القيمة الافتراضية هي false. ضبط هذا على false سيتسبب في تصدير Aspose.3D لمجموعة من شادر القمة/القطعة إذا كان ExportShaders مفعلاً.

```csharp
public bool UseCommonMaterials { get; set; }
```
