## Discreet3dsSaveOptions.HighPreciseColor property

إذا كان هذا true، سيستخدم ملف 3ds المُولد لونًا عالي الدقة، مما يعني أن كل قناة من الأحمر/الأخضر/الأزرق تكون في صيغة 32‑bit عائمة. وإلا سيستخدم الملف المُولد لونًا 24‑bit، حيث كل قناة تستخدم بايت 8‑bit. القيمة الافتراضية هي false، لأن ليس كل التطبيقات تدعم اللون عالي الدقة.

```csharp
public bool HighPreciseColor { get; set; }
```
