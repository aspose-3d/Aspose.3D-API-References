### clamp{#clamp}

| नाम | विवरण |
| --- | --- |
| clamp(val, min, max) | रेंज [min, max] में मान को क्लैंप करें |

 **Parameters:**

| नाम | प्रकार | विवरण |
| --- | --- | --- |
| val | Number | क्लैंप करने के लिए मान। |
| min | Number | न्यूनतम मान। |
| max | Number | अधिकतम मान। |

 **Result:**
Number
