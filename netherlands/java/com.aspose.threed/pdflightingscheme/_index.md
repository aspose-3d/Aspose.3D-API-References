LightingScheme specificeert de verlichting die op 3D-kunstwerken wordt toegepast.
## Velden

| Veld | Beschrijving |
| --- | --- |
| [ARTWORK](#ARTWORK) | Gebruikt de lichten die in de scène zijn gedefinieerd |
| [BLUE](#BLUE) | Drie blauwe oneindige lichten, geen omgevingsterm |
| [CAD](#CAD) | Drie grijze oneindige lichten en één licht bevestigd aan de camera, geen omgevingsterm |
| [CUBE](#CUBE) | Zes grijze oneindige lichten uitgelijnd met de hoofdassen, geen omgevingsterm |
| [DAY](#DAY) | Drie lichtgrijze oneindige lichten, geen omgevingsterm |
| [HARD](#HARD) | Drie grijze oneindige lichten, matige omgevingsterm |
| [HEADLAMP](#HEADLAMP) | Enkel oneindig licht bevestigd aan de camera, lage omgevingsterm |
| [NIGHT](#NIGHT) | Één geel, één aqua en één blauw oneindig licht, geen omgevingsterm |
| [NONE](#NONE) | Er worden geen lichten gebruikt. |
| [PRIMARY](#PRIMARY) | Één rood, één groen en één blauw oneindig licht, geen omgevingsterm |
| [RED](#RED) | Drie rode oneindige lichten, geen omgevingsterm |
| [WHITE](#WHITE) | Drie blauwgrijze oneindige lichten, geen omgevingsterm |
