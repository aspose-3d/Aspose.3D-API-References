## Klassen

| Klas | Beschrijving |
| --- | --- |
| [AnimationChannel](./animationchannel/) | Een kanaal wijst het componentveld van een eigenschap toe aan een set sleutelframereeksen |
| [AnimationClip](./animationclip/) | De animatieclip is een verzameling animaties. De scène kan een of meer animatieclips bevatten. |
| [AnimationNode](./animationnode/) | Aspose.3D's ondersteunt animatiehiërarchie, elke animatie kan worden samengesteld uit verschillende animaties en de keyframe-definitie van de animatie. [`AnimationNode`](../aspose.threed.animation/animationnode/) definieert de transformatie van een eigenschapswaarde in de loop van de tijd, een animatieknooppunt kan bijvoorbeeld worden gebruikt om de transformatie van een knooppunt of andere[`A3DObject`](../aspose.threed/a3dobject/) numerieke eigenschappen van object. |
| [BindPoint](./bindpoint/) | EEN[`BindPoint`](../aspose.threed.animation/bindpoint/)wordt meestal gemaakt op de eigenschap van een object, sommige eigenschapstypen bevatten meerdere componentvelden (zoals een Vector3-veld), [`BindPoint`](../aspose.threed.animation/bindpoint/) genereert een kanaal voor elk componentveld en verbindt het veld met een of meer keyframesequentie-instanties via de kanalen. |
| [Extrapolation](./extrapolation/) | Extrapolatie definieert hoe te handelen wanneer de bemonsterde waarde buiten het bereik valt dat is gedefinieerd door de eerste en laatste sleutelframes. |
| [KeyFrame](./keyframe/) | Een keyframe wordt voornamelijk gedefinieerd door een tijd en een waarde, voor sommige interpolatietypes wordt tangens/spanning/bias/continuïteit ook gebruikt om de uiteindelijke bemonsterde waarde te berekenen. Bemonsterde waarden in een niet-sleutelframe tijdpositie worden geïnterpoleerd door keyframes tussen de vorige en volgende keyframes Waarde voor/na het eerste/laatste keyframe worden berekend door de[`Extrapolation`](../aspose.threed.animation/extrapolation/) klasse. |
| [KeyframeSequence](./keyframesequence/) | De volgorde van sleutelframes, het beschrijft de transformatie van een bemonsterde waarde in de loop van de tijd. |
