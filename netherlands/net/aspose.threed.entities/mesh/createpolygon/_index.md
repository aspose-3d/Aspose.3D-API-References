Creëert een nieuwe polygoon met alle hoekpunten gedefinieerd in*indices* . Gebruik a.u.b. om polygoon hoekpunt voor hoekpunt te creëren[`PolygonBuilder`](../../polygonbuilder/) .

```csharp
public void CreatePolygon(int[] indices, int offset, int length)
```

| Parameter | Type | Beschrijving |
| --- | --- | --- |
| indices | Int32[] | Array van de polygoonindices, elke index wijst naar een besturingspunt dat de polygoon vormt. |
| offset | Int32 | De offset van de eerste polygoonindex |
| length | Int32 | De lengte van de indexen |
