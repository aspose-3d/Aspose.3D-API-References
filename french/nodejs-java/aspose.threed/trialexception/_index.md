## TrialException class

Cela est levé dans Scene.Open/Scene.Save lorsqu'aucune licence n'est appliquée.  Vous pouvez désactiver cette exception en définissant SuppressTrialException sur true.

| Nom | Description |
| --- | --- |
| getSuppressTrialException() | Définit cette valeur sur true pour supprimer l'exception d'essai en cas d'utilisation non licenciée, mais les restrictions ne seront pas levées. Pour lever les restrictions, veuillez utiliser une licence appropriée. Définir cette valeur sur true signifie également que vous êtes conscient des restrictions liées à l'absence de licence. |
