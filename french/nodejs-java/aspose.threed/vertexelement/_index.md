## VertexElement class

Classe de base des éléments de sommet.  Un type d'élément de sommet est identifié par VertexElementType.  Un VertexElement décrit comment l'élément de sommet est mappé sur une surface de géométrie et comment les informations de mappage sont organisées en mémoire.  Un VertexElement contient des normales, des UV ou d'autres types d'informations.  @hideconstructor
