## TriMesh.FromRawData method

Créer TriMesh à partir de données brutes

```csharp
public static TriMesh FromRawData(VertexDeclaration vd, byte[] vertices, int[] indices, 
    bool generateVertexMapping)
```

| Paramètre | Taper | La description |
| --- | --- | --- |
| vd | VertexDeclaration | Déclaration de vertex, doit contenir au moins un champ. |
| vertices | Byte[] | Les données de sommet d'entrée, la longueur minimale des sommets doit être supérieure ou égale à la taille de la déclaration de sommet |
| indices | Int32[] | Les indices triangulaires |
| generateVertexMapping | Boolean | Générer[`Vertex`](../../../aspose.threed.utilities/vertex) pour chaque sommet, ce qui n'est pas nécessaire uniquement pour la sérialisation/désérialisation. |

### Return_Value

La[`TriMesh`](../../trimesh) instance qui a encapsulé le tableau d'octets d'entrée.

### Remarques

Le TriMesh retourné ne copiera pas le tableau d'octets d'entrée pour les performances, les modifications externes sur le tableau seront répercutées sur cette instance.
