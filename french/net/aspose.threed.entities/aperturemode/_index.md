## ApertureMode enumeration

Modes d'ouverture de l'appareil photo. Le mode d'ouverture détermine les valeurs qui déterminent l'ouverture de l'appareil photo. Si le mode d'ouverture est HorizAndVert, Horizontal ou Vertical, le champ de vision est utilisé. Si le mode d'ouverture est FocalLength, alors la distance focale est utilisée.

### Valeurs

| Nom | Évaluer | La description |
| --- | --- | --- |
| HorizAndVert | `0` | Définissez les valeurs d'angle pour les paramètres horizontaux et verticaux. |
| Horizontal | `1` | Définissez uniquement l'angle horizontal. |
| Vertical | `2` | Définissez uniquement l'angle vertical. |
| FocalLength | `3` | Utiliser directement la distance focale. |
