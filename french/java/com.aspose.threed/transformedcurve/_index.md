```
public class TransformedCurve extends Curve
```

Un [TransformedCurve](../../com.aspose.threed/transformedcurve) donne à une courbe un placement en utilisant une matrice de transformation. Cela permet d'effectuer une transformation à l'intérieur d'un [TrimmedCurve](../../com.aspose.threed/trimmedcurve) ou d'un [CompositeCurve](../../com.aspose.threed/compositecurve).
