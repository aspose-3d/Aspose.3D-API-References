LightingScheme spécifie l'éclairage à appliquer à l'œuvre 3D.
## Champs

| Champ | Description |
| --- | --- |
| [ARTWORK](#ARTWORK) | Utilise les lumières définies dans la scène |
| [BLUE](#BLUE) | Trois lumières infinies bleues, aucun terme ambiant |
| [CAD](#CAD) | Trois lumières infinies grises et une lumière attachée à la caméra, aucun terme ambiant |
| [CUBE](#CUBE) | Six lumières infinies grises alignées avec les axes majeurs, aucun terme ambiant |
| [DAY](#DAY) | Trois lumières infinies gris clair, aucun terme ambiant |
| [HARD](#HARD) | Trois lumières infinies grises, terme ambiant modéré |
| [HEADLAMP](#HEADLAMP) | Une seule lumière infinie attachée à la caméra, terme ambiant faible |
| [NIGHT](#NIGHT) | Une lumière jaune, une lumière aqua et une lumière bleue infinie, aucun terme ambiant |
| [NONE](#NONE) | Aucune lumière n'est utilisée. |
| [PRIMARY](#PRIMARY) | Une lumière rouge, une lumière verte et une lumière bleue infinie, aucun terme ambiant |
| [RED](#RED) | Trois lumières infinies rouges, aucun terme ambiant |
| [WHITE](#WHITE) | Trois lumières bleu-gris infinies, aucun terme ambiant |
