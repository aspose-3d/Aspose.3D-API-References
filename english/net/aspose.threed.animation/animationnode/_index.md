## AnimationNode class

Aspose.3D's supports animation hierarchy, each animation can be composed by several animations and animation's key-frame definition. [`AnimationNode`](../animationnode) defines the transformation of a property value over time, for example, animation node can be used to control a node's transformation or other [`A3DObject`](../../aspose.threed/a3dobject) object's numerical properties.
