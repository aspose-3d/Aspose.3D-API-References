## TransformedCurve class

A [`TransformedCurve`](../transformedcurve) gives a curve a placement by using a transformation matrix. This allows to perform a transformation inside a [`TrimmedCurve`](../trimmedcurve) or [`CompositeCurve`](../compositecurve).
