## ITextureUnit interface

[`ITextureUnit`](../itextureunit) represents a texture in the memory that shared between GPU and CPU and can be sampled by the shader, where the [`Texture`](../../aspose.threed.shading/texture) only represents a reference to an external file. More details can be found https://en.wikipedia.org/wiki/Texture_mapping_unit
