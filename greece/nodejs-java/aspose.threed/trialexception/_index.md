## TrialException class

Αυτό προκαλείται στο Scene.Open/Scene.Save όταν δεν έχουν εφαρμοστεί άδειες. Μπορείτε να απενεργοποιήσετε αυτήν την εξαίρεση ορίζοντας το SuppressTrialException σε true.

### getSuppressTrialException{#getSuppressTrialException}

| Όνομα | Περιγραφή |
| --- | --- |
| getSuppressTrialException() | Ορίζει αυτό σε true για να καταστεί λυπημένη η εξαίρεση δοκιμής για χρήση χωρίς άδεια, αλλά οι περιορισμοί δεν θα αφαιρεθούν. Για να αφαιρεθούν οι περιορισμοί, παρακαλούμε χρησιμοποιήστε μια έγκυρη άδεια. Και το να το ορίσετε σε true σημαίνει επίσης ότι γνωρίζετε τους περιορισμούς χωρίς άδεια. |
