## VertexElementMaterial class

Ορίζει το δείκτη υλικού για τα καθορισμένα στοιχεία.  Ένας κόμβος μπορεί να έχει πολλαπλά υλικά, το VertexElementMaterial χρησιμοποιείται για την απόδοση διαφορετικών τμημάτων της γεωμετρίας σε διαφορετικά υλικά.
