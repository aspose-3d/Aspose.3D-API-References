## Transform class

Ένα transform περιέχει πληροφορίες που επιτρέπουν πρόσβαση στη μετάφραση/κλιμάκωση/περιστροφή του αντικειμένου ή στον πίνακα μετασχηματισμού με ελάχιστο κόστος. Αυτό χρησιμοποιείται από το local transform.  @hideconstructor
