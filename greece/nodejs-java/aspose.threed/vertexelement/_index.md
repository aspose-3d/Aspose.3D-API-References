## VertexElement class

Βασική κλάση των στοιχείων κορυφής.  Ένας τύπος στοιχείου κορυφής προσδιορίζεται από VertexElementType.  Ένα VertexElement περιγράφει πώς το στοιχείο κορυφής αντιστοιχίζεται σε μια γεωμετρική επιφάνεια και πώς οι πληροφορίες αντιστοίχισης διατάσσονται στη μνήμη.  Ένα VertexElement περιέχει Normals, UVs ή άλλου είδους πληροφορίες.  @hideconstructor
