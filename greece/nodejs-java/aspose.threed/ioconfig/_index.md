## IOConfig class

Διαμόρφωση IO για σειριοποίηση/αποσειριοποίηση. Ο χρήστης μπορεί να καθορίσει λεπτομερείς ρυθμίσεις όπως διαδρομή αναζήτησης εξαρτήσεων ή ρυθμίσεις σχετικές με τη μορφή εδώ  @hideconstructor


## Μέθοδοι

| Όνομα | Περιγραφή |
| --- | --- |
