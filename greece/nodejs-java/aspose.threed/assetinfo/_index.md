## AssetInfo class

Πληροφορίες του περιουσιακού στοιχείου. Οι πληροφορίες του περιουσιακού στοιχείου μπορούν να προσαρτηθούν σε μια Σκηνή. Η θυγατρική Σκηνή μπορεί να έχει το δικό της AssetInfo για να παρακάμψει τον ορισμό του γονέα.
