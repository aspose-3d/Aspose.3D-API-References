## TrialException.SuppressTrialException property

Το ορίζει σε true για να καταστείλει τη δοκιμαστική εξαίρεση για χρήση χωρίς άδεια, αλλά οι περιορισμοί δεν θα αρθούν. Για να άρετε τους περιορισμούς, χρησιμοποιήστε μια κατάλληλη άδεια χρήσης. Και το ορίζει σε true σημαίνει επίσης ότι γνωρίζετε την μη άδεια χρήσης περιορισμοί.

```csharp
public static bool SuppressTrialException { get; set; }
```
