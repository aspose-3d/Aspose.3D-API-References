## RvmSaveOptions.AttributePrefix property

Λαμβάνει ή ορίζει το πρόθεμα των χαρακτηριστικών που θα εξαχθούν, η εξαγόμενη ιδιότητα δεν θα περιέχει πρόθεμα, οι προσαρμοσμένες ιδιότητες με διαφορετικό πρόθεμα δεν θα εξαχθούν, η προεπιλεγμένη τιμή είναι 'rvm:'. Για παράδειγμα, εάν μια ιδιότητα είναι rvm:Refno =345, το εξαγόμενο χαρακτηριστικό θα είναι Refno = 345, το πρόθεμα έχει αφαιρεθεί.

```csharp
public string AttributePrefix { get; set; }
```
