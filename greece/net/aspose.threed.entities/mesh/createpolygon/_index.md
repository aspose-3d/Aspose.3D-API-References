## CreatePolygon(int[], int, int) {#createpolygon_3}

Δημιουργεί ένα νέο πολύγωνο με όλες τις κορυφές που ορίζονται σε*indices* . Για να δημιουργήσετε κορυφή πολυγώνου προς κορυφή, χρησιμοποιήστε[`PolygonBuilder`](../../polygonbuilder/) .

```csharp
public void CreatePolygon(int[] indices, int offset, int length)
```

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| indices | Int32[] | Πίνακας δεικτών πολυγώνου, κάθε ευρετήριο δείχνει σε ένα σημείο ελέγχου που σχηματίζει το πολύγωνο. |
| offset | Int32 | Η μετατόπιση του πρώτου δείκτη πολυγώνου |
| length | Int32 | Το μήκος των δεικτών |

### Παραδείγματα

```csharp

mesh.CreatePolygon(indices);
```
