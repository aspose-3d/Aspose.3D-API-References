### Παραδείγματα

ισούται με : Εάν όλοι οι δείκτες είναι έτοιμοι για χρήση,[`CreatePolygon`](../mesh/createpolygon/) προτιμάται, διαφορετικά`PolygonBuilder` θα ήταν καλύτερη επιλογή.

```csharp
PolygonBuilder builder = new PolygonBuilder(mesh);
builder.Begin();
builder.AddVertex(0);
builder.AddVertex(1);
builder.AddVertex(2);
Builder.End();
```

int[] indices = new int[] {0, 1, 2};
mesh.CreatePolygon(indices);
