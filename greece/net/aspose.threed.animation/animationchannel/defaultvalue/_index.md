## AnimationChannel.DefaultValue property

Λαμβάνει ή ορίζει την προεπιλεγμένη τιμή του καναλιού. Εάν ένα κανάλι δεν έχει συνδεδεμένες ακολουθίες καρέ-κλειδιού, η προεπιλεγμένη τιμή θα χρησιμοποιηθεί κατά την αξιολόγηση κινούμενων εικόνων. Ένα πραγματικό σενάριο: Η κίνηση κινείται μόνο τις συντεταγμένες x ενός κόμβου. δεν έχει αλλάξει, τότε η προεπιλεγμένη τιμή θα χρησιμοποιηθεί κατά την πλήρη αξιολόγηση της μετάφρασης.

public object DefaultValue { get; set; }
