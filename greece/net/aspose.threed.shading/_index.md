Όλες οι κλάσεις που σχετίζονται με τη σκίαση ορίζονται σε αυτόν τον χώρο ονομάτων.

## Τάξεις

| Τάξη | Περιγραφή |
| --- | --- |
| [LambertMaterial](./lambertmaterial/) | Υλικό για lambert shading model |
| [Material](./material/) | Το Υλικό καθορίζει τις απαραίτητες παραμέτρους για την οπτική εμφάνιση της γεωμετρίας. Το Aspose.3D παρέχει μοντέλο σκίασης για[`LambertMaterial`](../aspose.threed.shading/lambertmaterial/) ,[`PhongMaterial`](../aspose.threed.shading/phongmaterial/) και[`ShaderMaterial`](../aspose.threed.shading/shadermaterial/) |
| [PbrMaterial](./pbrmaterial/) | Υλικό για φυσική απόδοση με βάση το χρώμα albedo/μεταλλικό/τραχύτητα |
| [PbrSpecularMaterial](./pbrspecularmaterial/) | Υλικό για φυσική απόδοση με βάση το διάχυτο χρώμα/κατοπωλείο/γυαλάδα |
| [PhongMaterial](./phongmaterial/) | Υλικό για το μοντέλο σκίασης blinn-phong. |
| [ShaderMaterial](./shadermaterial/) | Ένα υλικό shader επιτρέπει την περιγραφή του υλικού με εξωτερική μηχανή απόδοσης ή γλώσσα shader. [`ShaderMaterial`](../aspose.threed.shading/shadermaterial/) χρήσεις[`ShaderTechnique`](../aspose.threed.shading/shadertechnique/) για να περιγράψετε τις λεπτομέρειες απόδοσης, και το καταλληλότερο θα χρησιμοποιηθούν σύμφωνα με την τελική πλατφόρμα απόδοσης. Για παράδειγμα,[`ShaderMaterial`](../aspose.threed.shading/shadermaterial/) Το παράδειγμα μπορεί να έχει δύο τεχνικές, η μία ορίζεται από το HLSL και η άλλη ορίζεται από το GLSL Σε πλατφόρμα χωρίς παράθυρο, το GLSL θα πρέπει να χρησιμοποιείται αντί για το HLSL |
| [ShaderTechnique](./shadertechnique/) | Μια τεχνική shader αντιπροσωπεύει μια υλοποίηση συγκεκριμένης απόδοσης. |
| [Texture](./texture/) | Αυτή η κλάση ορίζει την υφή από ένα εξωτερικό αρχείο. |
| [TextureBase](./texturebase/) | Βασική κλάση για όλες τις υφές σκυροδέματος. Η υφή καθορίζει την εμφάνιση και την αίσθηση μιας γεωμετρικής επιφάνειας. |
| [TextureSlot](./textureslot/) | Υποδοχή υφής[`Material`](../aspose.threed.shading/material/) , μπορεί να απαριθμηθεί μέσω της παρουσίας υλικού. |
