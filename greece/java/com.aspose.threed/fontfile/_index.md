Το αρχείο γραμματοσειράς περιέχει ορισμούς για γλύφους, αυτό χρησιμοποιείται για τη δημιουργία προφίλ κειμένου. **Παράδειγμα:** Ο παρακάτω κώδικας δείχνει πώς να δημιουργήσετε ένα 3D πλέγμα από κείμενο χρησιμοποιώντας το καθορισμένο αρχείο γραμματοσειράς.

```
var font = FontFile.fromFile("CascadiaCode-Regular.otf");
             var text = new Text();
             text.setFont(font);
             text.setContent("ABC");
             text.setFontSize(10);
             var linear = new LinearExtrusion(text, 10).toMesh();
             var scene = new Scene(linear);
             scene.save("test.stl");
```
